# Responsive Registration & Signup Page

A clean, modern, and split-screen responsive Signup Page built using semantic HTML5 and vanilla CSS3. The interface features a side-by-side layout with a vector illustration on one half and an interactive dark-themed form on the other.

## 🚀 Features

* **Split-Screen Layout**: Perfectly balanced 45/55 visual split using CSS Flexbox.
* **Modern UI Styling**: High-contrast dark signup dashboard with rounded aesthetic borders.
* **Interactive Form Elements**: Form inputs and custom buttons with smooth transition hover effects.
* **Semantic HTML**: Fully clean structure leveraging accessible labels and input types (`text`, `password`, `email`).
* **Clean Code Architecture**: Separated structures for layout (`index.html`) and presentation (`style.css`).

## 🛠️ Built With

* **HTML5**: Form architecture and structuring.
* **CSS3**: Layout design, Flexbox positioning, component styling, and UI animations.
* **Google Fonts / Monospace**: Typography styling.

## 🔧 Installation & Usage

To run this project locally, follow these simple steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/bharanibm/loginpage
   ```

2. **Navigate into the project directory**
   ```bash
   cd loginpage
   ```

3. **Open the project**
   * Double-click `index.html` to open it instantly in any modern web browser.
   * Alternatively, use VS Code and launch it via the **Live Server** extension.
